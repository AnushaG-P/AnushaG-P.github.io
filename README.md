Anusha Guruprasad Portfolio Website

How to use locally:
1. Open the folder in VS Code.
2. Install the Live Server extension if you do not already have it.
3. Right-click index.html and choose "Open with Live Server".

Responsive layout notes:
- Desktop: two-column hero section, four-column stats, three-column project cards.
- Tablet: stacked hero section, two-column cards.
- Mobile: hamburger navigation, stacked cards, full-width buttons, smaller spacing and headings.

Files:
- index.html
- styles.css
- script.js
- Anusha_Guruprasad_Resume.pdf

Deployment:
Upload the entire folder to GitHub Pages, Netlify, Vercel, or any static website host. Keep Anusha_Guruprasad_Resume.pdf in the same folder so the resume download button works.
